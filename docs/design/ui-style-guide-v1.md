# Agent Cluster 界面风格指南 v1

## 参考摘要

产品采用多智能体协作驾驶舱风格，由两个相互协调的界面组成：

- 聊天工作区：明亮的企业级即时通信布局，包含深色图标栏、会话列表、中央消息流，以及右侧智能体/任务面板。
- 协作图谱与工作流：深色霓虹指挥中心视图，使用深海军蓝背景、轻量网格纹理、发光卡片、彩色智能体泳道、方向箭头，以及紧凑的实时日志。

## 设计原则

- 始终保持协作状态可见：当前任务、在线智能体、执行进度和最新交接应成为首屏信号。
- 从视觉上区分模式：聊天界面保持明亮、偏操作；图谱/工作流界面保持深色、空间化、偏分析。
- 将智能体视为类似人的操作员：使用圆形头像标记、编号徽章、角色标签和按颜色区分的状态。
- 克制地使用发光效果来表达结构：仅用于活跃节点、路径箭头和主要状态。正文面板应保持清晰易读。

## 主题令牌

- 文本：`#0f1f3d`，弱化文本：`#60708f`，淡化文本：`#94a3b8`。
- 边框：`#dbe5f5`，选中边框：`#8bb8ff`。
- 成功：`#10b981`，警告：`#f59e0b`，危险：`#ef4444`。

深色驾驶舱外壳：

- 背景：`#020817`，面板：`rgba(7, 18, 38, 0.88)`。
- 面板边框：`rgba(80, 145, 255, 0.24)`。
- 主霓虹蓝：`#1d9bff`，青色：`#22d3ee`，绿色：`#19e58f`。

- 文本：`#eaf4ff`，弱化文本：`#8ba4c7`。
- 发光：`0 0 28px rgba(29, 155, 255, 0.28)`。

## 布局

- 桌面端聊天视图使用四个区域：图标栏、会话列表、中央时间线、右侧操作面板。
- 图谱/工作流视图保留图标栏和右侧日志面板，但将中央画布切换为深色驾驶舱样式。
- 浅色模式下主卡片使用 `8px` 圆角；深色画布节点在需要为发光效果留出更多呼吸空间时使用 `10px` 圆角。
- 阶段标签、头像标记、进度条和图标按钮等固定格式控件必须具有稳定尺寸，避免布局跳动。

## 组件

- 图标栏：深色垂直栏、蓝色激活指示器、白色线性图标，用户资料固定在靠近底部的位置。
- 会话列表：浅色卡片，包含头像组、群组/智能体数量标签、最后消息预览，以及选中态蓝色描边。
- 聊天消息：智能体头像位于左侧，发送者名称按智能体泳道着色，消息卡片带柔和阴影；用户消息右对齐，并使用浅蓝色背景。
- 智能体列表：紧凑卡片，包含头像、角色、状态标签、CPU/进度仪表，以及当前任务摘要。
- 任务步骤：垂直步骤条，包含编号状态和嵌套清单行。
- 图谱节点：头像居中放置在发光圆形框中；展示角色标题、智能体编号徽章、状态标签和按颜色区分的边框。
- 协作图谱：保留参考图中的深色驾驶舱布局：标题/任务行、`Agent 队列` 标题、五张固定位置的智能体卡片、绿色当前任务提示、连接线、实时状态表，以及右侧 `对话 / 消息日志` 面板，面板内包含消息卡片和底部输入框。使用项目现有调色板/令牌；不要引入新的颜色系统。
- 工作流视图：保留项目现有颜色和深色驾驶舱处理方式。布局遵循参考图：顶部运行时栏、左侧带状态图例的 `Agents` 栏、中央工作流画布、五张彩色阶段卡片、枢纽、输出区块、箭头连接线、底部工作流进度条，以及右侧实时 `对话 / 任务日志` 面板。

- 图谱和工作流画布：视觉内容层支持通过工具栏控件和鼠标滚轮在 60% 到 180% 之间缩放。缩放工具栏固定在画布外壳上；只有节点、连接线、枢纽和输出区块随缩放变化。

## 动效与反馈

- 悬停过渡：边框、阴影、背景和位移使用 160-220ms。
- 活跃的图谱/工作流元素可使用柔和的脉冲发光；避免大幅运动。
- 尊重 `prefers-reduced-motion`，禁用脉冲/浮动动画。

## 实现说明

- 使用 `styles.css` 中的 CSS 变量作为事实来源。
- 在工作区根节点上使用路由/模式类：
  - `workspace-shell mode-chat` 用于浅色聊天。
  - `workspace-shell mode-collaboration_graph` 和 `mode-workflow` 用于深色驾驶舱界面。
- 避免装饰性渐变光球。使用网格纹理、线条、阴影和状态颜色来营造层次。

## 聊天工作区规范

聊天页面遵循参考群聊布局。

- 根布局：`100vh` 固定应用外壳，包含四列：深色栏、会话列表、中央聊天工作区和右侧操作面板。
- 滚动：应用外壳、图标栏、会话列表和右侧面板都保持在视口内。只有中央聊天时间线在顶部标题和底部输入区之间滚动。
- 左侧栏：深海军蓝背景，使用明确的垂直布局：顶部品牌标识，中部是具备滚动安全边界的 `rail-nav` 功能组，当前用户区固定在底部。用户头像、姓名和在线状态必须作为一个 `rail-user` 组保持在一起。
- 会话列表：白色表面、主蓝色“新建会话”按钮、带搜索/筛选图标的搜索框、分段标签页，以及带蓝色边框和柔和蓝色背景的选中会话卡片。
- 顶部栏：左侧展示标题和摘要；右侧展示群组标签、头像堆叠、成员数量和视图模式按钮。群组标签/头像堆叠/成员数量用于打开完整智能体列表。不要在该顶部栏中展示搜索或更多操作。
- 时间线：消息使用左侧头像加白色圆角气泡；用户消息右对齐并使用浅蓝色气泡。气泡宽度需设置上限，保证可读性。
- 输入区：底部固定白色输入栏，包含四个工具图标（附件/图片/代码/提及）和一个蓝色圆形发送按钮。
- 右侧面板：智能体卡片使用肖像头像、角色文本、状态标签、当前任务、进度仪表，以及紧凑的能力/RAG 标签。任务执行步骤使用编号垂直步骤条。
- 图标系统：使用由 `@lucide/vue` 支撑的 `UiIcon.vue`；图标按钮保持 28-42px 正方形，线性图标保持 16-24px。
- 智能体肖像：使用 `AgentPortrait.vue`；颜色调性匹配共享智能体调色板（蓝、绿、紫、橙、青），并在聊天工作区中替代纯数字头像。
